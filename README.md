An AI-powered computer vision system that detects road hazards such as potholes, debris, and waterlogging from uploaded images. The project uses a YOLO-based object detection model for hazard identification and integrates a FastAPI backend to provide real-time inference through REST APIs.

**Key Features**:

• YOLO-based object detection model

• Hazard detection from user-uploaded road images

• FastAPI backend for image inference APIs

• Automated classification of road hazards

• Scalable SaaS-style architecture

**Tech Stack**:

Python, YOLO, OpenCV, FastAPI, Docker
